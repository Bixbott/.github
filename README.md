## ✨ Giới thiệu

Bixbott là một nền tảng Agent IDE thế hệ mới được thiết kế để hỗ trợ lập trình viên xây dựng ứng dụng nhanh hơn bằng sức mạnh của AI.

Nó kết hợp:

- 🤖 AI Coding Agent
- 🧠 Code Generation
- 🔍 Code Review
- 🚀 Deployment Assistant
- 📦 Project Scaffolding
- 🔄 GitHub Integration
- 🏷️ Multi-tag Task Management
- 👥 Personnel & Agent Assignment
